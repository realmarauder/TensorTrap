# Scan Reports

This folder contains TensorTrap security scan reports organized by machine.

## Folder Structure

```
reports/
├── local/      # Reports from local development machine
├── droplet/    # Reports from Digital Ocean droplet
└── README.md
```

## Report Formats

Each scan generates reports in multiple formats:
- `.txt` - Human-readable plain text
- `.json` - Machine-readable JSON for automation
- `.html` - Styled HTML report for viewing in browser
- `.csv` - Spreadsheet-compatible format for analysis
